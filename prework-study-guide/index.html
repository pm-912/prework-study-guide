Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • In the head element, usually metadata is stored that is not visible to the user.
  • HTML is the structure of the webpage, the foundation on which you add other elements

CSS

  • A margin indicates how much space we want around the outisde of an element.
  • A pagging indicates how much space we want around the content inside an element.
  • Three different ways to implement CSS are inline styling, internal style sheets, and external style sheets.
  • Inline affects the most specific and can be applied to augment specific elements.
  • An internal style sheet is placed in the head section, and affects all relevant elements on the page.
  • An external style sheet is a referenced document that can be linked through the head section, and affects all pertinent elements on the page.
  • An external style sheet can be used over multiple applications.

Git

  • git checkout main: switches to the main branch
